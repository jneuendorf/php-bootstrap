Easily generate bootstrap elements directly from PHP.
No more writing the same markup all over the place all the time.


The currently supported elements are buttons, alerts, breadcrumbs and modals. Below there are some demos showing the according PHP code and the output. Generally, you can choose between keyword-like arguments and positional arguments.

Button

Button::create(string label='', string kind='default', array classes=array(), string type='button', string size='', assoc_array attrs=array()) 
echo Button::create([
    ['label'] => 'button1'
])



echo Button::create([
    'button2'
])



echo Button::create([
    ['label'] => 'danger',
    ['kind'] => 'danger'
])



echo Button::create([
    'small info btn',
    'info',
    [
        'cumstom-class',
        'cumstom-class2'
    ],
    'button',
    'xs'
])



echo Button::create([
    ['label'] => 'big boy',
    ['size'] => 'lg'
])



echo Button::create([
    ['label'] => 'big boy2',
    ['classes'] => [
        'btn-lg'
    ]
])



echo Button::create([
    ['label'] => 'submit',
    ['type'] => 'submit'
])



echo Button::create([
    ['label'] => 'custom',
    ['attrs'] => [
        ['style'] => 'color: purple;'
    ]
])



Alerts

Alert::create(string kind, string content, bool dismissible, string id, classes=array(), assoc_array attrs=array()) 
echo Alert::create([
    'warning',
    'indismissible me'
])
indismissible me



echo Alert::create([
    ['kind'] => 'danger',
    ['content'] => '<strong>Dangerous</strong>',
    ['dismissible'] => true
])
Dangerous



echo Alert::begin(['kind' => 'success', 'dismissible' => true]); ?>
    This is a dismissible alert with a bit more complex HTML (which nobody wants to put into a PHP string).
    <a href="#" class="alert-link">This is a link with special markup also.</a><?php
echo Button::create('Check this out');
echo Alert::end();
This is a dismissible alert with a bit more complex HTML (which nobody wants to put into a PHP string). This is a link with special markup also.

Breadcrumbs

Breadcrumbs::create(array items=array(), array classes=array(), assoc_array attrs=array()) 
echo Breadcrumbs::create([
    ['Home'] => '#',
    ['Library'] => '#',
    ['Data'] => '#'
])
  1. Home
  2. Library
  3. Data



echo Breadcrumbs::create([
    ['items'] => [
        ['Homepage'] => 'http://www.google.com',
        'Page',
        'Subpage'
    ]
])
  1. Homepage
  2. Page
  3. Subpage



echo Breadcrumbs::create([
    ['Home'] => '#',
    ['No link'] => null
])
  1. Home
  2. No link



Modal

Modal::create(string title='', string body='', string footer='', string header='', string id='uid_'.uniqid(), array classes=array('fade'), assoc_array attrs=array(), bool initialize=false)
Note: 'classes' is set to '[]' to prevent that the 'fade' class is set. 
echo Modal::create([
    ['classes'] => []
])




echo Modal::create([
    ['header'] => null,
    ['footer'] => null,
    ['body'] => 'modal-body only',
    ['classes'] => []
])
modal-body only



echo Modal::create([
    ['title'] => 'My Modal',
    ['footer'] => null,
    ['body'] => 'custom title and no footer',
    ['classes'] => []
])

My Modal

custom title and no footer



echo Modal::create([
    'header' => null,
    'footer' => Button::create('cool', 'warning'),
    'body' => 'no header and custom button in footer',
    'classes' => []
])
no header and custom button in footer
echo Modal::create([
    ['body'] => 'initialized with JavaScript (thus it can be closed)',
    ['initialize'] => true,
    ['attrs'] => [
        ['data-backdrop'] => 'false'
    ]
])

initialized with JavaScript (thus it can be closed)



echo Modal::begin(['footer' => Button::create('Button'), 'classes' => []]);
echo Modal::header(); ?>
<div class="modal-body">
    <h5>
        Only the modal body is written in pure HTML.
    </h5>
</div>
<?php
echo Modal::footer();
echo Modal::end();

Only the modal body is written in pure HTML.
