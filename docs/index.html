Easily generate bootstrap elements directly from PHP.
No more writing the same markup all over the place all the time.


The currently supported elements are buttons, alerts, breadcrumbs and modals. Below there are some demos showing the according PHP code and the output. Generally, you can choose between keyword-like arguments and positional arguments.

Button

button(string label='', string kind='default', array classes=array(), string type='button', string size='', assoc_array attrs=array()) 
echo button([
    ['label'] => 'button1'
])



echo button([
    'button2'
])



echo button([
    ['label'] => 'danger',
    ['kind'] => 'danger'
])



echo button([
    'small info btn',
    'info',
    [
        'cumstom-class',
        'cumstom-class2'
    ],
    'button',
    'xs'
])



echo button([
    ['label'] => 'big boy',
    ['size'] => 'lg'
])



echo button([
    ['label'] => 'big boy2',
    ['classes'] => [
        'btn-lg'
    ]
])



echo button([
    ['label'] => 'submit',
    ['type'] => 'submit'
])



echo button([
    ['label'] => 'custom',
    ['attrs'] => [
        ['style'] => 'color: purple;'
    ]
])



Alerts

alert(string kind, string content, bool dismissible, string id, classes=array(), assoc_array attrs=array()) 
echo alert([
    'danger',
    'this looks dangerous'
])
this looks dangerous



echo alert([
    ['kind'] => 'warning',
    ['content'] => 'indismissible me (not)',
    ['dismissible'] => true
])
indismissible me (not)



echo alert_begin(['kind' => 'success', 'dismissible' => true]); ?>
This is a dismissible alert with a bit more complex HTML (which nobody wants to put into a PHP string).
<a href="#" class="alert-link">This is a link with special markup also.</a>
<?php
echo button('Check this out');
echo alert_end();
This is a dismissible alert with a bit more complex HTML (which nobody wants to put into a PHP string). This is a link with special markup also.

Breadcrumbs

breadcrumbs(array items=array(), array classes=array(), assoc_array attrs=array()) 
echo breadcrumbs([
    ['Home'] => '#',
    ['Library'] => '#',
    ['Data'] => '#'
])
  1. Home
  2. Library
  3. Data



echo breadcrumbs([
    ['items'] => [
        ['Homepage'] => 'http://www.google.com',
        'Page',
        'Subpage'
    ]
])
  1. Homepage
  2. Page
  3. Subpage



echo breadcrumbs([
    ['Home'] => '#',
    ['No link'] => null
])
  1. Home
  2. No link



Modal

modal(string title='', string body='', string footer='', string header='', string id='uid_'.uniqid(), array classes=array('fade'), assoc_array attrs=array(), bool initialize=false)
Note: 'classes' is set to '[]' to prevent that the 'fade' class is set. 
echo modal([
    ['classes'] => []
])




echo modal([
    ['header'] => null,
    ['footer'] => null,
    ['body'] => 'modal-body only',
    ['classes'] => []
])
modal-body only



echo modal([
    ['title'] => 'My Modal',
    ['footer'] => null,
    ['body'] => 'custom title and no footer',
    ['classes'] => []
])

My Modal

custom title and no footer



echo modal([
    'header' => null,
    'footer' => button('cool', 'warning'),
    'body' => 'no header and custom button in footer',
    'classes' => []
])
no header and custom button in footer
echo modal([
    ['body'] => 'initialized with JavaScript (thus it can be closed)',
    ['initialize'] => true,
    ['attrs'] => [
        ['data-backdrop'] => 'false'
    ]
])

initialized with JavaScript (thus it can be closed)



echo modal_begin(['footer' => button('Button'), 'classes' => []]);
echo modal_header(); ?>
<div class="modal-body">
    <h5>
        Only the modal body is written in pure HTML.
    </h5>
</div>
<?php
echo modal_footer();
echo modal_end();

Only the modal body is written in pure HTML.
